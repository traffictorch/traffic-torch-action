# 🚥 Traffic Torch SEO GEO UX Health Analysis Audit Toolkit

**One-click GitHub Action** that opens the full Traffic Torch interactive audit experience directly from your repository.

### Features
- Instant deep link to any Traffic Torch tool
- Pre-fills the URL automatically
- Full interactive mobile first experience, day/night mode, priority fixes, educational modules)
- Clean Job Summary with direct link
- Supports all major Traffic Torch tools

### How to use

1. Add this workflow to your repo: `.github/workflows/traffic-torch-audit.yml`
2. Go to **Actions** tab → "🚥 Traffic Torch SEO GEO UX Health Analysis Audit Toolkit"
3. Click **Run workflow**
4. Enter your URL and choose the tool
5. Click the generated link → full audit opens in your browser
